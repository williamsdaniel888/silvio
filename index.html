Silvio: Not Private, Not Public

The best defence against a state-sponsored counterfeiting industrial complex is multilateral sanctioning from the grassroots upwards. I'm going to be controversial and say Bitcoin should be privatized.

柔法の電気 / Technological Topics

Which language are you using to build Silvio?
Ultimately it should be platform-agnostic because it's fundamentally an expression of mathematical nature. The current source code is a tarball of .bf scripts hosted on the Gesellschaft repository (not private, not public), however we are actively exploring ports of the core implementation to WS and F#.

Are there existing implementations of similar ideas in other disciplines?
Randonautica is a similar effort, but applied to cartography rather than macroeconomics and relying on a quantum random number generator.

What is that exactly?
Through using quantum fluctuations to generate a map coordinate, you somehow change the fate of your life by visiting a place you would never have gone to.
